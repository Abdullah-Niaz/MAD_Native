# Simple Calculator App

## Overview
This is a basic Android calculator application that allows users to add two numbers. The application provides an intuitive interface where users can input two numbers, press the "Abdullah" button, and get the sum displayed in a result field.

## Features
- Simple and user-friendly UI
- Single-button operation for addition
- Real-time result display
- Properly styled input fields with borders
- Responsive design to fit different screen sizes
- Background color customization

## Installation
1. Clone the repository or download the project files.
2. Open the project in **Android Studio**.
3. Connect an Android device or launch an emulator.
4. Click **Run** to install and execute the application.

## Usage
1. Enter the first number in the first input field.
2. Enter the second number in the second input field.
3. Press the **"Abdullah"** button.
4. The sum of the two numbers will be displayed in the result field.

## Technologies Used
- **Java** (for application logic)
- **XML** (for UI design)
- **Android Studio** (for development & testing)

## Customization
- **Changing the Background Color**: Modify `android:background` in `activity_main.xml`.
- **Changing Button Name**: Edit `android:text` property of the Button element.
